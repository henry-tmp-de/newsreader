## 1. 用户画像构建与智能推荐模块

### 达成度

已达成。

### 已实现能力

- 动态用户画像：
  - 初始能力基线：用户等级（BEGINNER/INTERMEDIATE/ADVANCED）
  - 行为特征：阅读完成量、平均阅读时长、练习准确率、主动查词次数、近14天活跃度
  - 兴趣特征：注册兴趣 + 用户自主维护兴趣标签
- 推荐引擎：
  - 接入实时新闻库（已抓取并入库新闻）
  - 利用 LLM 生成摘要、关键词、难度标签
  - 结合文本可读性特征（句长、长词比例）与信息密度特征进行难度补全
  - 按 i+1 原则进行难度匹配并综合排序
- 新增用户反馈闭环（本次新增）：
  - 每篇推荐卡支持“感兴趣/不感兴趣”
  - 不感兴趣文章会被后续推荐过滤
  - 感兴趣/不感兴趣会沉淀为历史偏好，参与后续同类文章加权/降权

### 关键实现路径

- 后端接口：
  - GET /api/recommendations/profile
  - GET /api/recommendations/articles
  - PUT /api/recommendations/interests
  - POST /api/recommendations/feedback
- 前端页面：
  - 智能推荐页支持画像展示、兴趣编辑、反馈按钮与推荐流刷新
